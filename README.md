# Phreeqnics V1: Adaptive Reactive Transport Solver

## Overview
Phreeqnics is a powerful wrapper designed for reactive transport modelling by integrating the capabilities of FEniCS and PHREEQC. This tool primarily addresses the highly-advective diffusion-reaction transport using FEniCS and chemical reactions for equilibrium and kinetic using PHREEQC, making it highly suitable for several applications in environmental science and engineering.

## Key Features
- **Model Complexity**: Supports 1D and 2D problems with straightforward extension capabilities to 3D. 
- **User-Friendly**: Includes `AddInputParameters.py` for easy configuration of chemical, physical, and numerical settings.
- **Visualization**: Offers visualization options to analyze the simulation results effectively.
- **Flexibility**: Adapts to various complex transport approaches beyond the finite element method, including DG, and Adaptive Stabilized FEM (https://doi.org/10.1016/j.cma.2023.116285)

## Installation Requirements
- **Operating System**: Compatible with Linux and macOS.
- **Dependencies**:
  - FEniCS 2019 or later
  - Python 3.8+
  - phreeqpy

Ensure all dependencies are installed before running the software. For detailed installation instructions, refer to the official documentation of each dependency.
